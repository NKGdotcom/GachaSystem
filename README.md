# ガチャシステム　ソースコード
ソーシャルゲームなどにあるガチャシステムを作りました<br>
## 開発期間
3日間<br>
## 開発言語
C#<br>
## 使用ツール
Unity<br>
## システム概要
・石を5つ消費することで、ガチャを1回引くことができます<br>
・石を50個消費することで、ガチャを10回引くことができます<br>
・石を1個消費すると、キャラクターBOXを5枠拡張できます<br>
・石を10個消費すると、キャラクターBOXを50枠拡張できます<br>
・ガチャを1回引くとキャラクターが一体獲得することができ、真ん中に表示されます<br>
・ランダムな確率で様々なキャラクターがゲットできます<br>
・キャラクターの属性によってアイコンの色が変化します<br>
・ゲームの実行を終了しても、再読み込みすると、前の結果が呼び出されます<br>
